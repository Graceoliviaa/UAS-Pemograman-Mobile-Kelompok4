📱 WGS Project Management System - Kelompok 4

Sistem Informasi Manajemen Proyek, Klien, Tim, dan Dokumen internal yang dirancang khusus untuk memenuhi tugas Ujian Akhir Semester mata kuliah Pemrograman Mobile 1 di Universitas Teknologi Bandung. Aplikasi ini dibangun menggunakan platform Android Native dengan bahasa pemrograman Kotlin dan memanfaatkan Room Database untuk penyimpanan data lokal secara persisten.

🚀 Fitur Utama Aplikasi
1. Autentikasi Hak Akses: Log in aman menggunakan akun Project Manager.
2. Manajemen Proyek (CRUD): Menambah, melihat detail, memperbarui informasi, dan memantau status proyek perusahaan.
3. Manajemen Klien: Integrasi data klien, contact person (PIC), email, serta alamat perusahaan yang terhubung langsung ke proyek terkait.
4. Alokasi Tim Proyek: Menugaskan karyawan ke dalam tim proyek spesifik lengkap dengan penentuan role kerja.
5. Arsip Dokumen Proyek: Pengunggahan berkas digital pendukung proyek lengkap dengan pencatatan tanggal otomatis.
